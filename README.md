# 🤖 AMYRANG Bot

Bienvenue sur le dépôt officiel du bot **AMYRANG**. Ce bot est conçu en Python pour animer et gérer mon serveur Discord avec style.

---

## ✨ Fonctionnalités
*   **Musique :** Lecture de playlists et contrôle du volume.
*   **Modération :** Commandes de ban, kick et gestion des rôles.
*   **Fun :** Mini-jeux et interactions avec les membres.

---

## 🚀 Installation

1. Clone le projet :
   `git clone https://github.com/TON_NOM/AMYRANG-Bot.git`

2. Installe les dépendances :
   `pip install discord.py`

---

## ⚙️ Configuration

Crée un fichier `.env` à la racine du projet et ajoute ton token :
`DISCORD_TOKEN=ton_token_ici_secret`
